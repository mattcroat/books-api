# Books API

## Setup

Frontend:

- ⚛️ Next.js (frontend)
- ♻ React Query (managing server state)
- ⛵ Tailwind CSS (styles)

Backend:

- 🏃‍♀️ Express.js (backend)
- ✨ Swagger (API documentation)

## Instructions

```shell
# backend
cd server && npm start
```

```shell
# frontend
cd client && npm run dev
```
